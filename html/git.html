Команды Git

  • git init – инициализация нового репозитория.
  • git clone <url> – клонирование репозитория с удаленного сервера.
  • git add <файл> – добавление файла в индекс для коммита.
  • git commit -m "сообщение" – создание коммита с сообщением.
  • git push – отправка изменений на удаленный сервер.
